# CLI-VM-Manager
A W.I.P bash script that creates and manages QEMU VMs.

## Current Features
1. Create a VM
2. Start a VM

## W.I.P Features
1. Stop a VM
2. Delete a VM
3. VM Settings
4. Show Info / Performance Statistics 
5. GUI HTML Panel to simplify this instead of just CLI ; (Maybe will be a different repository for this but this is currently at our lowest priority.)
